File: cpuScheduler.c
Holds all the code for the RR, SRTF, and FCFs. Each of the scheduling mode has its own functions.

File: input.1
The file that the cpuScheduler.c code will read from and get the process information from.

Compile program by going directly to the directory and typeing in
"gcc -o proj2 cpuScheduler.c"
and type in the commands
"proj2 input.1 FCFS" or 
"proj2 input.1 RR (#)" [# == quantumTime of user choosing] or
"proj2 input.1 SRTF"
